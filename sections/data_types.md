# Типы данных

## Цель
Познакомиться с основными типами данных:
- Целые числа
- Числа с плавающей запятой
- Символы и строки
- Логические переменные
- Перечислимые типы
- Именованные константы
- Массивы
- Собственные типы данных (псевдонимы)
